# Sanctuary Resort - Mock Website

This is a high-fidelity mock website for the Sanctuary Eco-Luxe Resort portfolio project.
Built with Next.js 14 (App Router) using Vanilla CSS Modules/Globals.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Pages Implemented

- **Homepage**: `/` - Hero section, intro, and destinations list.
- **Destination Detail**: `/destinations/1` - Editorial layout for "Kyoto Retreat".
- **Booking Flow**: `/book` - Simple checkout form.

## Design System
- Fonts: Playfair Display (Headings), Inter (Body).
- Colors: Deep Olive, Warm Alabaster, Sage Green.
